01
O único passo manual
Clone e abra — o Claude faz o resto
⏱ 1 minuto

Este é o único comando de todo o processo. Cole no terminal, aperte Enter, e o Claude abre na pasta certa e começa o setup automaticamente.

Terminal — copie e cole esta linha
# Clone + entra na pasta + abre o Claude (tudo em um comando) $ git clone https://github.com/zxmarketingdigital/agente-ia-vendas.git && cd agente-ia-vendas && claude
✅
Quando o Claude abrir, ele lê o CLAUDE.md automaticamente e já começa dizendo: "Oi! Vou criar seu agente de vendas agora..." — sem você precisar pedir nada.
⚠️
Precisa ter instalado: Python 3.9+, Node.js 18+, Docker e Git. Se faltar algo, o Claude detecta e mostra como instalar.
